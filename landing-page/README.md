# 3d.london Recruitment Landing Page

This is a personalized landing page built for a specific job application at **One (Bedford)**.

## Project Structure

- **Astro**: Static site generator for performance.
- **Tailwind CSS**: Styling framework.
- **Assets**: All visuals (images/videos) are located in `public/assets/`.

## Development

1.  `npm install`
2.  `npm run dev` (Starts local server at localhost:4321)
3.  `npm run build` (Generates production files in `dist/`)

## Asset Management

Currently, the project uses **placeholder SVGs** generated for layout purposes.
To finalize the site, replace the files in `public/assets/img` and `public/assets/video` with the real campaign assets keeping the same filenames.
